# Qareeb Mobile App (React Native)

This project is now configured as an Expo React Native application.

## Prerequisites

- Node.js 18+
- Android Studio Emulator or Expo Go app on your phone

## Run the app

1. Install dependencies

npm install

2. Start Expo

npm run start

3. Open on device/emulator

- Press a in terminal for Android
- Scan the QR code with Expo Go

## Useful scripts

- npm run start
- npm run android
- npm run ios
- npm run web
- npm run lint
